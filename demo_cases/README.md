# Demo Cases

Готовые учебные кейсы для показа проекта.

- `*_minimal.json` — короткий сценарий быстрого демо.
- `*_extended.json` — расширенный сценарий с большим числом признаков.

Кейсы можно:

- загрузить через Streamlit как файл;
- открыть в режиме `JSON` и вставить содержимое вручную;
- использовать для CLI через промежуточное преобразование в CSV или как ориентир по полям.
